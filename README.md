## Overview

This project demonstrates effective stakeholder communication in a data analytics workflow.

## Scenario

As a data professional for the National Park Service (NPS), I was tasked with writing two emails about a machine learning project designed to predict future visitation at the most visited national parks.

The emails were tailored to two different audiences:

- A new data professional joining the project
- A public relations writer responsible for communicating with the public

The goal was to communicate the same project while adjusting the level of technical detail to meet each audience's needs.
